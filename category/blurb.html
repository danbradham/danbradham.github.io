First Post

Fri 22 November 2013

So I began cleaning up my github repos and realized that I wanted a more permanent solution for blogging. Python and Pelican provided me with a great solution; write my posts in markdown and generate static html pages. It may not be the most elegant solution, but, it sure feels great to be liberated from blogger!!

I plan to write primarily about Python and Autodesk's ever-surprising Maya. I'll also post some of my artwork here and there. It's been awhile since I've worked on any personal art projects, but, I hope to change that in the near future.
